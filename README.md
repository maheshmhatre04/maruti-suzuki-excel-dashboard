# Maruti Suzuki Sales Dashboard (Excel)

## Project Overview

This project is an interactive Sales Performance Dashboard built in Microsoft Excel to analyze sales data for Maruti Suzuki.

## Key Features

* Interactive slicers for Month, Region, and Car Model
* KPIs: Total Sales, Revenue, Top Model, Monthly Growth
* Trend analysis using charts
* Business insights for decision-making

## Tools Used

* Microsoft Excel
* Pivot Tables
* Charts
* Slicers
* KPI Formulas

## Files Included

* Dashboard.xlsx
* dashboard-preview.png
* dataset.csv

## Key Learnings

* Dashboard design
* KPI creation
* Data analysis
* Business reporting
